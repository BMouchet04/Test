# Simulation de connexion (devoir phishing)

Page unique : `index.html` — maquette de la première étape de connexion Google.

## Utilisation

Ouvrir `index.html` dans un navigateur ou via GitHub Pages.

Saisir une adresse e-mail fictive, cliquer **Suivant** → redirection vers la page de connexion Google officielle.

Simulation pédagogique uniquement. N'utilisez pas de vrais identifiants.
